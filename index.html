SmartCart — Configuration

Firebase déjà configuré ✅

SmartCart est connecté à Firebase. Si vous rencontrez un problème, vérifiez les règles Firestore ci-dessous.

1
Créer un projet Firebase
Allez sur console.firebase.google.com → "Ajouter un projet" → nom "smartcart" → désactiver Analytics.
2
Activer Authentication
Authentication → Sign-in method → Activer : Google, Apple (Apple ID requis), Email/Password.
3
Activer Firestore
Firestore Database → Créer → Mode production → Choisir une région européenne (eur3).
4
Copier la config
Paramètres du projet → Vos applications → "Web" → Copier l'objet firebaseConfig ci-dessous.
Collez votre configuration Firebase :
🎁 Plan gratuit Firebase (Spark)
50K lectures/jour · 20K écritures/jour · 1 GB stockage · Auth illimités. Largement suffisant pour une famille ou un logement partagé.
📋 Règles Firestore requises
rules_version = '2';
service cloud.firestore {
  match /databases/{db}/documents {
   match /users/{uid} {
    allow read, write: if request.auth.uid == uid;
   }
   match /households/{code} {
    allow read, write: if request.auth != null;
   }
  }
}
Copiez ces règles dans Firebase Console → Firestore → Règles. Sinon l'app retournera "permission denied".
